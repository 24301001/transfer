# 修改文件清单

## 构建与配置

- `pom.xml`：基于用户提供的 Spring Boot 3.2.0 / Java 17 POM，新增 `spring-boot-starter-data-redis`。
- `src/main/resources/application.properties`：新增 Redis、JWT issuer/audience/clock skew、滑块验证码、验证码次数与发送冷却配置；敏感密钥改用环境变量。
- `docker-compose.redis.yml`：Redis 7.4 本地容器配置。
- `.env.example`、`start-local.ps1.example`：本地启动环境变量示例。
